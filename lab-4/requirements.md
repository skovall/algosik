## 1. Сокращатель ссылок

### Описание задачи

Необходимо реализовать упрощённый сервис сокращения ссылок.

Система должна уметь:

- сохранять длинную ссылку
- выдавать ей короткий код
- по короткому коду возвращать исходную ссылку

Например:

- длинная ссылка: `https://example.com/articles/python-basics`
- короткий код: `a1b2`
- результат хранения: `a1b2 -> https://example.com/articles/python-basics`

### Обязательная часть

Реализовать:

- добавление новой ссылки
- получение длинной ссылки по короткому коду
- проверку существования короткого кода
- вывод всех сокращённых ссылок
